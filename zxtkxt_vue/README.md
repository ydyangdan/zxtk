# 基于SpringBoot的高并发选课系统

## Project setup
```
npm install
```
## 技术选型

前端：vue3、element plus

后端：SpringBoot

中间件：Redis、Druid

数据库：MySQL

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
